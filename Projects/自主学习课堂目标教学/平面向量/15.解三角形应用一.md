>[!question] 三角形的面积公式
>$$S=\frac{1}{2}ab\sin C$$结合正弦定理和外接圆，推导一下其他的变异形式

>[!tip] 提示
>$\begin{flalign*}&S=\frac{1}{2}=2R^2\sin A\sin B\sin C\\
>&S=\frac{abc}{4R}\end{flalign*}$

>[!question] 题目1
>三角形$\triangle ABC$中，$a=2,A=60^{\circ}$,求：
>$(1)b+c$的最大值
>$(2)\triangle ABC$面积的最大值

>[!tip]- 解析1
>由正弦定理：
>设外接圆半径为$R$，则
$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = \dfrac{2}{\sin 60^\circ} = \dfrac{2}{\frac{\sqrt{3}}{2}} = \dfrac{4}{\sqrt{3}}=2R.$
>且 $B + C = 120^\circ$。
>
>(1)$b = \frac{4}{\sqrt{3}} \sin B, \quad c = \frac{4}{\sqrt{3}} \sin C$
$b + c = \frac{4}{\sqrt{3}} [\sin B + \sin (120^\circ - B)]$
>用和差化积：
>$\sin B + \sin (120^\circ - B) = 2 \sin \left( \frac{B + 120^\circ - B}{2} \right) \cos \left( \frac{B - (120^\circ - B)}{2} \right)$
>$= 2 \sin 60^\circ \cos (B - 60^\circ)$
>$= \sqrt{3} \cos (B - 60^\circ)$
>故此
$b + c = \frac{4}{\sqrt{3}} \cdot \sqrt{3} \cos (B - 60^\circ) = 4 \cos (B - 60^\circ)$
>$\cos (B - 60^\circ)$ 最大为 $1$，此时 $B = 60^\circ$, $C = 60^\circ$，即 $b = c$, $B = C = 60^\circ$，最大值 $4$。
>
>(2)$\sin B\sin C=\dfrac{1}{2}[\cos(B-C)-\cos(B+C)]$
>$=\dfrac{1}{2}\cos(2B-\frac{2\pi}{3})+\dfrac{1}{2}$
>$0<B<\dfrac{2\pi}{3},\therefore -\dfrac{2\pi}{3}<2B<\dfrac{2\pi}{3}$
>$\sin B\sin C\le\dfrac{3}{4}$
>$S=\dfrac{1}{2}bc\sin A=\sqrt{3}R^2\sin B\sin C\le\sqrt{3}$

>[!tip]- 解析2
>(1)$a^2=b^2+c^2-2bc\cos A=\Rightarrow b^2+c^2-bc=4$
>整理可得$(b+c)^2-3bc=4$
>即$(b+c)^2-4=3bc\le 3\times(\dfrac{b+c}{2})^2$
>即可解得$b+c\le 4$当且仅当$b=c=2$时，等号成立。
>（2）$bc+4=b^2+c^2\ge2bc\Rightarrow bc\le4,$
>$\therefore S\le\sqrt{3}$

<span style='color:red'>解法一更通用</span>

>[!question] 题目2
>$\text{在}\Delta ABC\text{中,角}A,B,C\text{的对边分别是}a,b,c,\text{若}b=3,2a\mathrm{cos}C+2c\mathrm{cos}A=3a,\text{则}a=\underline{\quad}.$

>[!tip]- 解析
>因为$2a\cos C+2c\cos A=3a$，由正弦定理，可得$2\sin A\cos C+2\sin C\cos A=3\sin A$．
>所以$2\sin(A+C)=3\sin A$，又因为$A+C=\pi-B$，所以$\sin(A+C)=\sin B$，
>所以$2\sin B=3\sin A$，又由正弦定理，可得$2b=3a$，即$a=\frac{2}{3}b$，
>因为$b=3$，所以$a=2$．


>[!question] 题目3
>已知$\triangle ABC$中,$AB=4,AC=6,\angle BAC=60^{\circ},$作$\angle BAC$的角平分线与$BC$交于点$D$,求$AD$的长度

>[!tip]- 解析
>$S_{\triangle ABC}=S_{\triangle ABD}+S_{\triangle ACD}$可得
>$\frac{1}{2}AB\cdot AC\sin\angle BAC=\frac{1}{2}AB\cdot AD\sin\angle BAD+\frac{1}{2}AD\cdot AC\sin\angle DAC$
>$4\times6\times \sin60^{\circ}=4\times AD\sin30^{\circ}+6\times AD\sin30^{\circ}$
>$\therefore AD=\dfrac{12\sqrt{3}}{5}$


>[!question] 题目4
>在$\Delta ABC$中，内角$A,B,C$所对的边分别为$a,b,c,\angle BAC=120^{\circ},AD$为$\angle BAC$的角平分线，且$AD=2.$
(1)若$\sin B=2$sin$C$,求$a$的大小；
(2)当$b+c$取得最小值时，求$\triangle ABC$的面积

>[!tip]- 解析
>(1) 因为$\sin B=2\sin C$, 由正弦定理得$b=2c$,
因为$\angle BAC$的角平分线交$BC$于点$D$, 所以$\angle BAD=\angle CAD=60^\circ$,
由$S_{\triangle ABC}=S_{\triangle ADB}+S_{\triangle ACD}$, 得$\frac{1}{2}bc\sin \angle BAC=\frac{1}{2}c\cdot AD\sin \angle BAD+\frac{1}{2}b\cdot AD\sin \angle CAD$,
则$\frac{1}{2}\times 2\times b\sin 60^\circ+\frac{1}{2}\times 2\times c\times \sin 60^\circ=\frac{1}{2}\times bcsin120^\circ$,
即$bc=2c+2b$, 所以$c=3$, $b=6$,
在$\triangle ABC$中, 由余弦定理得$a^2=b^2+c^2-2accos120^\circ=36+9-2\times 3\times 6\times (-\frac{1}{2})=63$,
即$a=\sqrt{63}=3\sqrt{7}$;
>(2) 由$S_{\triangle ABC}=S_{\triangle ADB}+S_{\triangle ACD}$,
得$\frac{1}{2}bc\sin \angle BAC=\frac{1}{2}c\cdot AD\sin \angle BAD+\frac{1}{2}b\cdot AD\sin \angle CAD$,
得$\frac{1}{2}\times 2\times b\sin 60^\circ+\frac{1}{2}\times 2\times c\times \sin 60^\circ=\frac{1}{2}\times bcsin120^\circ$,
化简得$bc=2c+2b$, 即$\frac{2}{b}+\frac{2}{c}=1$,
所以$b+c=(b+c)(\frac{2}{b}+\frac{2}{c})=2+\frac{2b}{c}+\frac{2c}{b}+2\geq 2\sqrt{\frac{2b}{c}\cdot \frac{2c}{b}}+4=8$,
当且仅当$b=c=4$时等号成立, $b+c$取得最小值,
此时, $\triangle ABC$面积为$\frac{1}{2}\times b\times c\times \sin 120^\circ=\frac{1}{2}\times 4\times 4\times \frac{\sqrt{3}}{2}=4\sqrt{3}$.
