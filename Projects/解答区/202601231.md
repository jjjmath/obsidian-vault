>[!success] 题目1
>$幂函数f(x)过点(8,2),g(x)=f(|x|),则不等式g(2a-1)<g(a+2)的解集为()$
>- [ ] $A.(3,+\infty)$
>- [ ] $B.(-\infty,3)$
>- [x] $C.(-\frac{1}{3},3)$
>- [ ] $D.(-\infty,-\frac{1}{3})\cup(3,+\infty)$

>[!help] 解析
>$f(x)=x^\alpha,8^\alpha=2,\therefore \alpha=\frac{1}{3}$
>$所以f(x)在[0,+\infty)上递增$
>$g(x)=f(|x|)为偶函数在[0,+\infty)上递增$
>$g(2a-1)<g(a+2)可得|2a-1|<|a+2|$
>$(2a-1)^2<(a+2)^2解得-\frac{1}{3}<x<3$
