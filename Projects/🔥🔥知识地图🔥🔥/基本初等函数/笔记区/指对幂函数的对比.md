# 📊 幂函数、指数函数、对数函数性质对比表
![[Pasted image 20251212103452.png]]

## 一、基本对比表

| 性质维度       | **幂函数** $( y = x^\alpha ) (α为常数)$                                                                                  | **指数函数** $( y = a^x ) (a>0,a≠1)$             | **对数函数** $( y = \log_a x ) (a>0,a≠1)$                        |
| ---------- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------- | ------------------------------------------------------------ |
| **定义域**    | 因α而异：<br>• α为正整数：R<br>• α为负整数：x≠0<br>• α为分数：分母偶→x≥0<br>• α为无理数：x>0                                                 | 全体实数 R                                       | 正实数 (0, +∞)                                                  |
| **值域**     | 因α而异：<br>• α>0：[0,+∞)或R<br>• α<0：(0,+∞)或(-∞,0)∪(0,+∞)                                                              | (0, +∞)                                      | 全体实数 R                                                       |
| **过定点**    | (1, 1)（所有幂函数都过）                                                                                                    | (0, 1)（所有指数函数都过）                             | (1, 0)（所有对数函数都过）                                             |
| **单调性**    | 第一象限内：<br>• α>0：增函数<br>• α<0：减函数<br>• α=0：常函数y=1                                                                   | • a>1：在R上单调递增<br>• 0<a<1：在R上单调递减             | • a>1：在(0,+∞)上单调递增<br>• 0<a<1：在(0,+∞)上单调递减                   |
| **奇偶性**    | 由α决定：<br>• α为奇→奇函数<br>• α为偶→偶函数<br>• α非整→非奇非偶                                                                      | **非奇非偶**                                     | **非奇非偶**                                                     |
| **图像特征**   | • 都过(1,1)<br>• 在第一象限都有图像<br>• 形状由α决定                                                                               | • 在x轴上方<br>• 以x轴(y=0)为渐近线<br>• 平滑曲线          | • 在y轴右侧<br>• 以y轴(x=0)为渐近线<br>• 平滑曲线                          |
| **渐近线**    | α<0时：坐标轴为渐近线                                                                                                       | y = 0（x轴）                                    | x = 0（y轴）                                                    |
| **与坐标轴交点** | • 与y轴：可能无交点(α<0)<br>• 与x轴：可能多个交点                                                                                   | 与x轴无交点（y>0）                                  | 与y轴无交点（x>0）<br>与x轴交点(1,0)                                    |
| **特殊形式**   | • α=1：y=x（直线）<br>• α=2：y=x²（抛物线）<br>• α=3：y=x³（立方曲线）<br>• $α=\frac{1}{2}：y=\sqrt x$<br>• $α=-1：y=\frac{1}{x}（反比例）$ | • $a=e：y=e^x$（自然指数）<br>• $a=10：y=10^x$（常用指数） | • $a=e：y=lnx$（自然对数）<br>•$a=10：y=lgx$（常用对数）<br>•$a=2：y=log₂x$ |

## 二、**a>1 与 0<a<1 对比表**

| 比较项         | **指数函数** $( y = a^x )$                    | **对数函数** $( y = \log_a x )$              |
| ----------- | ----------------------------------------- | ---------------------------------------- |
| **a>1 时**   | • 单调递增<br>• $x→-∞, y→0$<br>• $x→+∞, y→+∞$ | • 单调递增<br>• x→0⁺, y→-∞<br>• $x→+∞, y→+∞$ |
| **0<a<1 时** | • 单调递减<br>• x→-∞, y→+∞<br>• x→+∞, y→0     | • 单调递减<br>• x→0⁺, y→+∞<br>• x→+∞, y→-∞   |
| **共性**      | 都过定点：(0,1)                                | 都过定点：(1,0)                               |

## 三、**特殊关系与互化表**

| 关系类型      | 具体表现                                      | 说明                    |
| --------- | ----------------------------------------- | --------------------- |
| **互为反函数** | $( y = a^x ) ↔ ( y = \log_a x )$          | 图像关于直线 \( y = x \) 对称 |
| **互化公式**  | $( a^x = N ) ⇔ ( x = \log_a N )$          | 指数式 ↔ 对数式             |
| **函数变换**  | $( y = a^{x+h} + k )$                     | 指数函数的平移<br>h:左右, k:上下 |
|           | $( y = \log_a (x-h) + k )$                | 对数函数的平移               |
| **复合形式**  | $( y = \log_a (b^x) = x \cdot \log_a b )$ | 对数+指数复合               |
|           | $( y = a^{\log_a x} = x )$                | 互为反函数性质               |

## 四、**记忆口诀**

**幂函数**：
> “一三象限看奇偶，正值递增负值掉；  
> 分数小心定义域，所有都过(1,1)点。”

**指数函数**：
> “底大于1往上爬，底在0到1往下滑；  
> 永在x轴上方挂，定点(0,1)不离它。”

**对数函数**：
> “底大于1右上升，底在0到1右下降；  
> y轴是堵墙，定点(1,0)不能忘。”

## 五、**常见易错点**

| 函数类型     | 易错点                                          | 正确理解         |
| -------- | -------------------------------------------- | ------------ |
| **幂函数**  | 认为 $( x^{-2} )$ 定义域为R                        | 实际为 {x\|x≠0} |
|          | $认为 ( x^{\frac{1}{3}}) 与 ( \sqrt[3]{x} ) 不同$ | 两者等价，定义域均为R  |
| **指数函数** | 认为底数a可为负数                                    | a>0且a≠1      |
|          | 忽略 $( a^0 = 1 )$ 的运用                         | 所有指数函数过(0,1) |
| **对数函数** | 忽略真数>0的条件                                    | 定义域(0,+∞)    |
|          | 认为 $( \log_a (x+y) = \log_a x + \log_a y )$  | 错误！没有此性质     |
