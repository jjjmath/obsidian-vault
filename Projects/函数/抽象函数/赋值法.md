>[!example] **题目1**
>
>$已知函数f(x)的定义域为R，对任意的实数x,y满足$
>$f(x+y)=f(x)+f(y)-1$
>$当x<0时，f(x)>1,则下列结论错误的是$
>- [ ]  $f(0)=1$
>- [x] $f(x)-1为偶函数$
>- [ ] $f(x)为R上的增函数$
>- [ ] $若f(a-6)+f(a^2)>2,则a的取值范围是(-3,2)$
>[[赋值法1]]


>[!example] **题目2**
>
>$已知函数f(x),对任意的x,y\in R,都有f(x+y)+f(x-y)=2f(x)f(y)$
>$且，f(1)=\frac{1}{2},则下列说法正确的是$
>- [ ] $f(0)=0$
>- [ ] $f(x)为奇函数$
>- [x] $f(x)>-1$
>- [ ] $f(2025)=1$
>[[赋值法2]]
