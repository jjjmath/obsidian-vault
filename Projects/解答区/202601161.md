>[!success] 题目1
>$已知函数f(x)=x^2+mx-2m+n,若非空集合A=\{x|f(x)\le0\}$
>$B=\{x|f(f(x)+2)\le4\},且A=B,下列说法正确的是()$
>- [ ] $A.n的取值与m有关$
>- [x] $B.n为定值$
>- [ ] $C.0\le m\le2\sqrt2$
>- [x] $D.0\le m\le2\sqrt5-2$

>[!help] 解析
>$\because A,B为非空集合$
>$复合函数方式对待$
>$令f(x)+2=t$
>$f(f(x)+2)\le4即f(t)\le4$
>$设f(t)\le4的解集为[a,b]$
>$f(b)=4$
>$则a\le f(x)+2\le b\Rightarrow a-2\le f(x)\le b-2$
>$依题意，a-2\le f(x)\le b-2与f(x)\le0的解集相同$
>$\color{red}翻译$
>$b-2=0,且f(x)的最小值f(x)_{\min}\ge a-2$
>$b=2\Rightarrow f(b)=f(2)=4+2m-2m+n=4$
>$n=0$
>$f(x)=x^2+mx-2m$

>[!help] 

>[!tip] 特殊情形
>$A、B可否为单元素集合$
