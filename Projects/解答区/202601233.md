>[!success] 题目3
>$已知定义域为R的函数f(x)满足：\forall x_1,x_2\in R,x_1\neq x_2,都有\frac{f(x_1)+x_1-f(x_2)-x_2}{x_1-x_2}>0$
>$且f(3)=0,若f(2m+1)>2-2m,则实数m的取值范围是()$
>- [ ] $A.(-\infty,0)$
>- [ ] $B.(0,+\infty)$
>- [ ] $C.(-\infty,1)$
>- [x] $D.(1,+\infty)$

>[!help] 解析
>$令g(x)=f(x)+x$
>$则\frac{f(x_1)+x_1-f(x_2)-x_2}{x_1-x_2}>0可化为\frac{g(x_1)-g(x_2)}{x_1-x_2}>0$
>$\therefore g(x)=f(x)+x在R上为增函数$
>$g(3)=f(3)+3=3$
>$f(2m+1)>2-2m可化为f(2m+1)+2m+1>3即g(2m+1)>g(3)$
>$\therefore 2m+1>3,m>1$
