# 第 1 页

隐零点
解释：零点不可求，
但是可以做两件事情
(1)判断零点所在区间
(2)建⽴零点得⽅程
举例
熟悉问题
证明：ex − ln x − 1 > 0
第⼀步：确定最终⽬标
不⽤放缩法的时候
x > 0
设f(x) = ex − ln x − 1(⽬标依然是想画草图)
求导-----
f ′(x) = ex − 1
x = xex − 1
x
为了画图，判断f ′(x)的符号
第⼆步转化+计算
完成对f ′(x)的符号判断
注意转化
∵ x > 0,所以只需判断xex − 1的符号
设g(x) = xex − 1
第三步：研究g(x)

# 第 2 页

g′(x) = (1 + x)ex > 0
得到g(x)在(0, +∞)单调递增
⼜因为g(0) = −1 < 0, g(1) = e − 1 > 0
上⾯两个条件确定了g(x)的草图
即存在x0 ∈ (0, 1)使得g(x0) = 0
确定隐零点的范围
范围可以调整，根据题⽬对精确度的要求
x0ex0 = 1
确定隐零点的等量关系式
第四步
根据导函数的符号，判断原函数
(0, x0), g(x) < 0, (x0, +∞), g(x) > 0
(0, x0), f ′(x) < 0, (x0, +∞), f ′(x) > 0
从⽽得到f(x)的图像变化特征
回到最开始的⽬标：画草图
