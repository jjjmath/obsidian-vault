>[!success] 题目2
>$函数f(x)的定义与为R，且对任意的实数x，都有f(x)=f(x-2)-f(4-x),且f(0)=2$
>$下列说法正确的是()$
>- [x] $A.f(x)为偶函数$
>- [x] $B.f(x)为周期函数，且12是函数的周期$
>- [x] $C.f(4)=-1$
>- [ ] $D.\displaystyle\sum_{k=1}^{25} f(2k)=2$

>[!解析]
>$f(x)=f(x-2)-f(4-x),f(0)=2$
>$令x=2,f(2)=f(0)-f(2)\Rightarrow f(2)=1$
>$令x=4,f(4)=f(2)-f(0)=-1$


>[!continue]
>$x+2替换x，f(x+2)=f(x)-f(2-x)\Rightarrow f(x)=f(x+2)-f(2-x)$
>$-x替换x,f(-x)=f(2-x)+f(2+x)$
>$\therefore f(-x)=f(x),f(x)为偶函数$

>[!continue]
>$f(x)=f(x-2)-f(4-x),x+4替换x$
>$f(x+4)=f(x+2)-f(-x)$
>$f(x+4)+f(-x)=f(x+2)$
>$\because f(-x)=f(x)$
>$f(x+4)+f(x)=f(x+2)$
>$f(x+6)+f(x+2)=f(x+4)\Rightarrow f(x+2)=f(x+4)-f(x+6)$
>$f(x+6)=-f(x)\Rightarrow T=12$
>$\because f(x+6+6)=-f(x+6)=f(x)$

>[!continue]
>$f(0)=2,f(2)=1,f(4)=-1$
>$f(6)=f(4)-f(2)=-2$
>$f(8)=f(6)-f(4)=-1$
>$f(10)=f(8)-f(6)=1$
>$f(12)=f(10)-f(8)=2$
>
