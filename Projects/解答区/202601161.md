>[!success] 题目1
>$已知函数f(x)=x^2+mx-2m+n,若非空集合A=\{x|f(x)\le0\}$
>$B=\{x|f(f(x)+2)\le4\},且A=B,下列说法正确的是()$
>- [ ] $A.n的取值与m有关$
>- [x] $B.n为定值$
>- [ ] $C.0\le m\le2\sqrt2$
>- [x] $D.0\le m\le2\sqrt5-2$

>[!help] 解析
>$\because A,B为非空集合$
>$复合函数方式对待$
>$令f(x)+2=t$
>$f(f(x)+2)\le4即f(t)\le4$
>$设f(t)\le4的解集为\{x|a\le x\le b\}$
>$f(b)=4$
>$则a\le f(x)+2\le b\Rightarrow a-2\le f(x)\le b-2$
>$依题意，a-2\le f(x)\le b-2与f(x)\le0的解集相同$
>$\color{red}翻译$
>$b-2=0,且f(x)的最小值f(x)_{\min}\ge a-2$
>$b=2\Rightarrow f(b)=f(2)=4+2m-2m+n=4$
>$n=0$
>$所以B正确$

>[!help] continue
>$A\neq\varnothing,f(x)=x^2+mx-2m$
>$\Delta=m^2+8m\ge0\Rightarrow a\le-8或a\ge0$

>[!help] continue
>$设f(t)\le4的解集为[a,b]\Rightarrow f(x)=4的两个根为a、b=2$
>$韦达定理\begin{cases}a+2=-m\\2a=-2m-4\end{cases}\Rightarrow a=-m-2$

>[!help] continue
>$f(x)_{\min}\ge a-2\Rightarrow f(-\frac{m}{2})=-\frac{m^2}{4}-2m\ge -m-4\Rightarrow m^2+4m-16\le0$
>$-2\sqrt5-2\le m\le 2\sqrt5-2$

>[!help] 综上所述
>$0\le m\le2\sqrt5-2$
