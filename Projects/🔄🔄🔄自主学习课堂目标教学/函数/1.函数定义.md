>[!important] 目标
>$1.通过会议初中函数的含义，充分认识和理解函数的定义$
>$2.熟悉新的函数符号f(x)及基本的函数求值$


>[!tip] - [ ] 任务一
>$看课本引例$

>[!定义]
>$设A，B为两个非空数集，对于\forall x\in A按照某种对应关系f$
>$在集合B中都存在唯一确定的元素y与之对应，这种对应关系叫做函数$
>$记法f：A\longmapsto B$
>$x的取值集合叫定义域A$
>$y的取值集合C叫值域，C\subseteq B$
>$记作y=f(x)$

>[!success] 初中所学函数的定义域和值域


| 函数                      | 定义域             | 值域                                  |
| ----------------------- | --------------- | ----------------------------------- |
| $y=ax+b(a\neq0)$        | R               | R                                   |
| $y=\frac{k}{x}(k\neq0)$ | $\{x\|x\neq0\}$ | $\{y\|y\neq0\}$                     |
| $y=ax^2+bx+c(a\neq0)$   | R               | $\{y\|y\ge\frac{4ac-b^2}{4a}\},a>0$ |
|                         |                 | $\{y\|y\le\frac{4ac-b^2}{4a}\},a<0$ |
>[!success] 题目1
>$已知f(x)=x^2+2x,求f(2),f(a),f(a+1),f(f(x))$

>[!success] - [ ] 任务二
>$区间的概念$
>$a<x<b表示方法开区间(a,b)$
>$a\le x\le b表示方法闭区间[a,b]$
>$x>a表示为(a,+\infty)$
>$x\le a表示为半开半闭区间(-\infty,a]$
>$\infty读作\text{“无穷大”}$

>[!help] 题目2
>$求函数的定义域用区间表示$
>$(1)y=\frac{1}{x}$
>$(2)y=\sqrt x$
>>[!答案]-
>>$(1)(-\infty,0)\cup(0,+\infty)$
>>$(2)[0,+\infty)$
