# 题干

在 $\triangle ABC$ 中，角 $A,B,C$ 所对的边分别为 $a,b,c$，且 $a \cos B = b \cos A$。
（1）判断 $\triangle ABC$ 的形状；
（2）若 $a=2, c=1$，求 $\triangle ABC$ 的面积。

---

# 答案

（1）等腰三角形（$a=b$）；（2）$\dfrac{\sqrt{15}}{4}$

---

# 解析

## 步骤1：化简条件
由正弦定理，$a=2R\sin A$, $b=2R\sin B$，代入 $a\cos B = b\cos A$ 得：
$2R\sin A \cos B = 2R\sin B \cos A$  
$\Rightarrow \sin A\cos B - \cos A\sin B = 0$  
$\Rightarrow \sin(A-B)=0$  
$\Rightarrow A=B$，故 $a=b$，为等腰三角形。

## 步骤2：求面积
由（1）知 $a=b=2$，又 $c=1$，用余弦定理求 $\cos C$：  
$\cos C = \dfrac{a^2+b^2-c^2}{2ab} = \dfrac{4+4-1}{8} = \dfrac{7}{8}$  
$\sin C = \sqrt{1-\cos^2 C} = \dfrac{\sqrt{15}}{8}$  
面积 $S = \dfrac12 ab\sin C = \dfrac12 \times 2\times 2\times \dfrac{\sqrt{15}}{8} = \dfrac{\sqrt{15}}{4}$
