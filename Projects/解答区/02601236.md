>[!help] 题目6
>$已知定义在R上的函数f(x)满足:f(-x)+f(x)=0,f(2-x)=f(x),且f(x)在$
>$[-1,1]内单调递增，则()$
>- [x] $A.f(8)<f(5.5)<f(-3)$
>- [ ] $B.f(-3)<f(5.5)<f(8)$
>- [ ] $C.f(5.5)<f(-3)<f(8)$
>- [ ] $A.f(8)<f(-3)<f(5.5)$

>[!success] 解析
>$由题可得$
>$\color{blue}f(x)为奇函数，且关于x=1对称$
>$\begin{cases}f(-x)=-f(x)\\f(2-x)=f(x)\end{cases},可得f(2-x)=-f(-x)$
>$\color{red}即f(2+x)=-f(x)$
>$\color{blue}f(x+4)=-f(x+2)=f(x)所以函数的周期为4$
>$f(8)=f(0)$
>$f(5.5)=f(1.5)=f(2-0.5)=f(0.5)$
>$f(-3)=f(1)$
>$f(0)<f(0.5)<f(1)$
