## 💡 **使用技巧**
1. **一致性**：建立自己的图标规则表
2. **位置固定**：统一放在开头：`⭐ 核心概念.md`
3. **组合使用**：`🔥⭐ 双重重点.md`
4. **颜色意义**：
   - 🔴 红色：紧急/警告
   - 🟡 黄色：进行中/注意
   - 🟢 绿色：完成/安全
   - 🔵 蓝色：信息/冷静
   - 🟣 紫色：创意/特殊

## 🎨 **高级玩法**
```markdown
# 我的图标索引（放在日记或主页）
## 优先级
- 🔴 = 今日必须完成
- 🟡 = 本周内完成  
- 🟢 = 有空时处理

## 类型
- 📚 = 永久知识
- 📝 = 临时笔记
- 💡 = 灵感碎片

## 状态
- 🚀 = 活跃项目
- ⏸️ = 暂停中
- ✅ = 已完成存档
```
